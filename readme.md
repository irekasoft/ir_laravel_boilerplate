# iRekaSoft Laravel Boilerplate

Laravel (5.8.x) + React(16.2.x) setup for web application.

First Laravel setup:

`$ cp env.example .env`

`$ php artisan key:generate`

`$ composer install`



Setup React

`$ yarn install`

`$ yarn watch`


### Using Valet 

`$ valet link <yoursite>`

### How to setup auth 

Setup .env file to setup integration with mysql.

`$ php artisan migrate `
